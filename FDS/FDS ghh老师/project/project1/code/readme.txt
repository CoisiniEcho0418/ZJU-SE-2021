Tips1: 多次使用功能2（自动化测试）输出test结果后，再使用功能4（输出平均数）就可以获得多次test结果的平均数。但是注意，如若要调用功能3（修改配置）且还想之后再调用功能4（输出平均数），请保证在调用功能2（自动化测试）之前先调用功能3来修改配置，否则输出的平均数会有问题。  [这是由于程序逻辑设计的目的是为了获得更精确的平均数，所以对每一个算法的单次执行时间是先加总之后再统一除以K值的来获得的]

Tips2: 请保证每次输入选项时都是整数类型的数字，如果输入字符等非数字类型，程序将无法运行。

Tips3: 在绝大多数情况下，程序能够正常编译并且运行成功。如果实在编译不了，可以把此目录下的 "project1.exee" 文件后缀名改为exe，这样就可以获得一个可直接运行的exe文件。

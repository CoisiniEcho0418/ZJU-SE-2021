Tips1: 该项目实现了两种算法，分别为check_dijkstra1和check_dijkstra2，若想测试两种算法的准确性，可以在源程序的43-47行，通过注释掉其中一个算法来切换程序运行选定的算法。

Tips2: 该项目实现的Dijstra算法是适用于无向图的，若想切换成有向图，需要修改源代码的第35行，将edge[y][x]删掉。需要注意的是，在报告中提供的测试用例均为无向图测试用例，如果想测试有向图，需要自己提供，我提供的测试用例可能有些不适用。

Tips3: 在绝大多数情况下，程序能够正常编译并且运行成功。如果实在编译不了，可以把此目录下的 "project3.exee" 文件后缀名改为exe，这样就可以获得一个可直接运行的exe文件。

Tips4: 该项目默认所有的输入都是符合格式规范的，所以没有提供格式检错机制，如果你提供的测试用例有格式问题，程序可能将无法正常运行（无法输出正确结果）
